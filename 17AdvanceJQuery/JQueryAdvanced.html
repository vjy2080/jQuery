In jQuery, traversing means moving through or over the HTML elements to find, filter or select a particular or entire element.

parent() it gives parent element of specified selector.
$(selector).parent();

parents() it gives all ancestor elements of the specified selector.
$(selector).parents();

parentsUntil()it gives all ancestor elements between specified selector and arguments.
$(selector).parentsUntil(selector, filter element)
$(selector).parentsUntil(element, filter element)

offsetParent() it gives the first positioned parent element of specified selector.
$(selector).offsetParent();

closest() it gives the first ancestor of the specified selector.
$(selector).closest(selector);
$(selector).closest(selector, context);
$(selector).closest(selection);
$(selector).closest(element);

children()it gives the children of each selected elements, optionally filtered by a selector.
$(selector).children();

find() it gives descendant elements of specified elements, filtered by a selector, jQuery object, or element.
$(selector).find('selector to find');

siblings() it gives all siblings of the specified selector.
$(selector).siblings();

next()it gives the next sibling element of the specified selector.
$(selector).next();

nextAll() it gives all next sibling elements of the specified selector.
$(selector).nextAll();

nextUntil()it gives all next sibling elements between specified selector and arguments.
$(selector).nextUntil();

prev() it gives the previous sibling element of the specified selector.
p$(selector).prev(selector);
$(selector).prev()

prevAll()it gives all previous sibling elements of the specified selector.
$(selector).prevAll(selector, filter element)
$(selector).prevAll(element, filter element)

prevUntil()it gives all previous sibling elements between specified selector and arguments.
$(selector).prevUntil(selector, filter element)
$(selector).prevUntil(element, filter element)

Filtering first()it gives the first element of the specified selector.
$(selector).first();

Filtering last()it gives the last element of the specified selector.
$(selector).last();

Filtering eq()it gives an element with a specific index number of the specified selector.
$(selector).eq(index);
$(selector).eq( indexFromEnd );

Filtering filter() it remove/detect an elements that are matched with specified selector.
$(selector).filter(selector)
$(selector).filter(function)
$(selector).filter(selection)
$(selector).filter(elements)

Filtering has()it gives all elements that have one or more elements within, that are matched with specified selector.
$(selector).has(selector);

Filtering is() it checks if one of the specified selector is matched with arguments.
.is( selector )
.is( function )
.is( selection )
.is( elements )

map()Pass each element in the current matched set through a function, producing a new jQuery object containing the return values
.map( callback )

slice()it selects a subset of specified selector based on its argument index or by start and stop value.
$(selector).slice(start, end );
$(selector).slice(start);

slice()it selects a subset of specified selector based on its argument index or by start and stop value.
$(selector).slice(start, end );
$(selector).slice(start);

Miscellaneous Traversing add()it add all elements to set of matched elements to manipulate them at the same time.
$(selector).add(selector to add);

addBack() it add the previous set of elements on the stack to the current set, optionally filtered by a selector.
$(selector).addBack();

andSelf() Deprecated 1.8 which is alias of addBack().
$(selector).addSelf();

contents()it gives all direct children, including text and comment nodes, of the specified selector.
$(selector).contents();

not()it gives all elements that do not match with specified selector.
$(selector).not(selector);
